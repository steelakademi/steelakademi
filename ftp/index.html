🚧 Kurulum Aşamasında
⚠️ Dikkat: Bu dosyalar FTP ile yüklenmiştir. Uygulamanın çalışması için sunucuda kurulum yapılması gerekmektedir.

📋 Kurulum Adımları:

  1. Node.js ve npm kurulu olduğundan emin olun
  2. Terminal/SSH ile sunucuya bağlanın
  3. npm install --production komutunu çalıştırın
  4. npx prisma generate komutunu çalıştırın
  5. npx prisma db push komutunu çalıştırın
  6. .env dosyasını production ortamına göre düzenleyin
  7. npm start ile uygulamayı başlatın
✅ Dosyalar Hazır: Tüm gerekli dosyalar yüklenmiştir. Detaylı kurulum rehberi için KURULUM_REHBERI.md dosyasını inceleyin.

🔧 Teknik Destek: Kurulum sırasında sorun yaşarsanız logları kontrol edin.

📚 Dokümantasyon: README.md dosyasında proje hakkında detaylı bilgi bulabilirsiniz.
